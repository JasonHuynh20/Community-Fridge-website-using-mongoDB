Files:

fridge-router.js - the javascript file that defines the routes for the application
server.js - the javascript file for the running the server
config.js - the javascript file that contains the data needed to connect to the database 

fridgeModel.js - the javascript file that contains the schema for the fridge documents
itemModel.js - the javascript file that contains the schema for the items documents
typeModel.js - the javascript file that contains the schema for the type documents

data folder - contains javascript files for loading up the data


Compilation Instructions: In the command line, type "Node server.js" then click enter. This will start the server.

Execution Instructions: After compilations, type "http://localhost:8000/" into the address bar followed by various 
other things to test routes.
